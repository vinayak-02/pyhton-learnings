Line Plot¶

a type of plot which displays information as a series of data points called “markers” connected by straight lines.

In this type of plot, we need the measurement points to be ordered (typically by their x-axis values).

This type of plot is often used to visualize a trend in data over intervals of time - a time series.

Scatter Plot¶

This type of plot shows all individual data points.

they aren’t connected with lines. Each data point has the value of the x-axis value and the value from the y-axis values.

This type of plot can be used to display trends or correlations. In data analytics, it shows how 2 variables compare.

Box plot¶

Box plot, also called the box-and-whisker plot is a way to show the distribution of values based on the five-number summary:

  • minimum
  • first quartile
  • median
  • third quartile
  • maximum

Bar chart¶

A bar chart represents categorical data with rectangular bars. Each bar has a height corresponds to the value it represents. It’s useful when we want to compare a given numeric value on different categories.

Pie chart¶

A pie chart is a circular plot, divided into slices to show numerical proportion.

They are widely used in the business world. However, many experts recommend to avoid them. The main reason is that it’s difficult to compare the sections of a given pie chart. Also, it’s difficult to compare data across multiple pie charts. In many cases, they can be replaced by a bar chart.
