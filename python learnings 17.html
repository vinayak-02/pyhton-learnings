1. Making Decisions with if (Conditional statement)¶

The if statement conditionally runs instructions inside the body of the if statement. The code is only run if the condition evaluates to True.

1-1 The if Condition¶

The statement that follows the if keyword will be evaluated to True or False. While you're coding remember that the statement you write must be asking a question of some form. ```

In [1]:
grade=71
if grade>=70:
   print("passed")

1-2 if and else¶

The else statement lets you run code when something is not true. An else can only be used in combination with an if. Together they let you run one alternative no matter what the condition is.

When you have and if/else one of the alternatives will always be run depending on the condition.

Notice that no condition follows the else. Whatever is in the else body is only run when the if condition is False and so it doesn't need any more information. Enter the example in the next cell:

In [2]:
grade=88
if grade>=93:
    print('A')
elif grade>=90:
    print('A-')
elif grade>=87:
    print('B+')

1-3 Asking Multiple Questions with elif¶

Using if and else we've been able to simplify the code a bit but there's more we can do. When a program has to respond to a question with more than two alternatives the elif statement allows us to ask another question. The elif is short for "else if" and must follow an initial if statement and will only be run if the initial condition is False.

Here are the rules to remember when you construct if, elif, and else statements:

  1. They must start with if
  2. elif is optional and you can have as many as you like.
  3. else is optional and must be at the end.
  4. In an if, elif, else structure only one alternative will execute.
In [3]:
grade=55
if grade>=70:
   print("passed")
else:
    print("failed")

2. Try and Except¶

try statement enable exception handling.

Especially, the try statement is useful to handle errors.

The try statements try clause begins with keyword try, followed by a colon (:) and a block of instruction that might raise exceptions.

A try clause may be followed by one or more except clauses that immediately follow the try clauses block.
